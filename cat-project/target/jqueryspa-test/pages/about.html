About

My name is Arik (@lastboy13 Github(lastboy)) the creator of CatJS framework that is an open source automation framework. I used it for testing this application.

This project was created while I visited the JQuery Chicago conference site and I thought to myself, why not to come up with a nice skeleton project that can be used by others.

The initial project is based on TodoMVC AngularJS + RequireJS project, I just modified it according to my needs. Feel free to use it and if you have any questions or comment you know where to find me...
